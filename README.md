# Воркхаб театра Т.А.М.

Рабочее PWA-приложение камерного театра-лаборатории Т.А.М.

Текущая версия подключена к Supabase и содержит общий и личный вход, роли и права доступа, общую «Копилку материалов», загрузку файлов, комментарии, реакции и корзину с хранением материалов в течение 30 дней.

## Локальный запуск

```bash
pnpm install
pnpm run dev
```

Для запуска нужны переменные `VITE_SUPABASE_URL` и `VITE_SUPABASE_PUBLISHABLE_KEY` в локальном файле `.env.local`. Значения секретов не хранятся в Git.

Публикация выполняется автоматически через GitHub Pages после отправки изменений в ветку `main`.
