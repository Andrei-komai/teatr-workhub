# Воркхаб театра Т.А.М.

Рабочее PWA-приложение камерного театра-лаборатории Т.А.М.

Текущая версия содержит локальный прототип «Копилки материалов». Облачная синхронизация, личные роли, загрузка файлов и push-уведомления подключаются на следующем этапе.

## Локальный запуск

```bash
npm install
npm run dev
```

Пароль прототипа по умолчанию: `tam`. Его можно переопределить переменной `VITE_HUB_PASSWORD`.
